## Wizardry: Proving Grounds of the Mad Overlord Save Editor

This is a savegame editor for the game "Wizardry: Proving Grounds of the Mad Overlord" (1981). The editor is designed to work with the MS-DOS release from WizardryArchives.

Based off of the original command-line utility by [gualberti](https://github.com/gualberti), found [here](https://github.com/gualberti/WizardrySaveEditor). My main contributions were translation from Italian to English, as well as a GUI.
